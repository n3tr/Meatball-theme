12APR2015 Development

Hubot เป็น Chatbot ชนิดหนึ่งที่ถูกสร้างขึ้นโดย GitHub เพื่อเป็นผู้ใช้อำนวยความสะดวกให้กับเราผ่านทางแชทต่างๆ เช่นหากเราต้องการแผนที่, ลิ้งค์วิดีโอ, หรือรูปภาพต่างๆ เราสามารถสั่ง Hubot ผ่านทางแชทได้ทันที (ChatOps)

นอกจากนี้ Hubot ยังมี Script การทำงานอื่นๆ ที่เราสามารถติดตั้งเพื่อเพิ่มความสามารถให้กับ Hubot ให้สามารถมาทำให้อะไรให้กับเราได้มากขึ้นอีกด้วย เช่น การบันทึกข้อความ, การสั่งรัน Job บน Jenkins, หรือมากกว่าขึ้นเราสามารถสั่ง Deploy และ Rollback ผ่านทาง Hubot ด้วยก็ยังได้

เบื้องหลัง Hubot คือ Node.js/CoffeeScript โดยตัว Hubot ได้ถูกออกแบบมาให้สามารถทำงานบน Heroku ได้เป็นอย่างดี และรองรับบริการแชทที่หลากหลาย เช่น Camfire, Hipchat, Slack, หรือแม้กระทั่ง IRC

เริ่มติดตั้ง Hubot

เนื่องจาก Hubot เป็น Node.js Application ดั้งนั้นเราจำเป็นต้องมี Node Package Manager (NPM) เพื่อใช้ในการติดตั้ง Hubot หากยังไม่มีสามารถดาวโหลดได้จากเว็บไซต์ http://nodejs.org

“ That means you can use 3rd party which call advertisingIdentifier but make sureAdSupport.framework not include in your binary even you remove it from your project/workspace. (because some 3rd party load AdSupport into binary in compile time) ”

จากนั้นให้ไปสร้าง Application บน Heroku เพื่อใช้ในการส่ง Hubot ขึ้นไป (หากใครไม่เคยใช้ Heroku คร่าวๆ คือ Heroku เป็น Application Hosting ประเภท Platform as a service (PAAS) โดยที่เราสามารถนำ Code ของเราขึ้นไปรันได้ทันที โดยหากเป็น App เล็กที่ทรัพยากรไม่มากนั้นสามารถใช้งานได้ฟรี)
